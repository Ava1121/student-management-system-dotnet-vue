## 一、系统目标

本实验基于实验 1-3 的技术积累，实现一个前后端分离的学生管理系统。通过本次实验，学生应掌握前后端分离架构的核心思想与实现方式，熟练使用 .NET Web API 构建 RESTful 接口，掌握 EF Framework Code First 开发模式，理解并应用 Autofac IOC 容器实现依赖注入，学会使用前端框架与后端 API 进行交互。

系统目标包括：
1. 实现学生信息的增删改查功能
2. 实现课程信息的增删改查功能
3. 实现成绩信息的增删改查功能
4. 实现用户登录认证和权限管理
5. 提供友好的用户体验和系统性能

## 二、系统业务需求

### 总体功能

学生管理系统是一个面向学校管理人员和学生的信息管理系统，主要用于管理学生的基本信息、课程信息和成绩信息。系统分为管理员和学生两种角色，分别具有不同的功能权限。

### 子功能模块

1. **用户认证模块**
   - 管理员登录：使用管理员账号密码登录系统，获取管理员权限
   - 学生登录：使用学生ID和密码登录系统，获取学生权限
   - Token 验证：使用 JWT Token 进行接口认证和权限验证

2. **学生管理模块**
   - 学生信息查询：管理员可以查看所有学生的信息
   - 学生信息添加：管理员可以添加新的学生信息
   - 学生信息修改：管理员可以修改现有学生的信息
   - 学生信息删除：管理员可以删除学生信息
   - 学生个人信息查看：学生可以查看自己的个人信息

3. **课程管理模块**
   - 课程信息查询：管理员可以查看所有课程的信息
   - 课程信息添加：管理员可以添加新的课程信息
   - 课程信息修改：管理员可以修改现有课程的信息
   - 课程信息删除：管理员可以删除课程信息
   - 学生课程查看：学生可以查看自己选修的课程

4. **成绩管理模块**
   - 成绩信息查询：管理员可以查看所有学生的成绩信息
   - 成绩信息添加：管理员可以添加新的成绩信息
   - 成绩信息修改：管理员可以修改现有成绩的信息
   - 成绩信息删除：管理员可以删除成绩信息
   - 学生成绩查看：学生可以查看自己的成绩信息

### 3.1 系统功能结构

系统采用前后端分离架构，分为前端和后端两个主要部分：

**后端部分**：
- 数据访问层：使用 EF Core Code First 实现数据访问
- 业务逻辑层：实现核心业务逻辑
- 接口层：提供 RESTful API 接口
- 认证授权层：实现 JWT Token 认证和权限管理

**前端部分**：
- 路由层：实现页面路由管理
- 组件层：实现可复用的 UI 组件
- 页面层：实现系统的各个页面
- 服务层：封装与后端 API 的交互

### 3.2 数据库设计

#### 实体关系

系统主要包含以下实体：

1. **学生实体**：包含学生的基本信息，如学号、姓名、年龄、邮箱等
2. **课程实体**：包含课程的基本信息，如课程ID、课程名称、学分等
3. **成绩实体**：包含学生的成绩信息，如成绩ID、学生ID、课程ID、分数等
4. **用户实体**：包含用户的登录信息，如用户名、密码、角色等

实体之间的关系：
- 一个学生可以选修多门课程，一门课程可以被多个学生选修（多对多关系，通过成绩表关联）
- 一个学生可以有多个成绩记录，一个课程可以有多个成绩记录
- 一个用户对应一个学生或管理员

### 5.3 登录模块实现

#### 功能描述
登录模块负责验证用户身份，生成 JWT Token，并根据用户角色跳转到相应的页面。

#### 核心代码实现

**前端登录页面**
```vue
<template>
  <div class="login-container">
    <el-card class="login-card" shadow="hover">
      <template #header>
        <div class="login-title">学生管理系统登录</div>
      </template>
      
      <el-form :model="loginForm" :rules="rules" ref="loginFormRef" label-width="80px">
        <el-form-item label="用户名" prop="username">
          <el-input v-model="loginForm.username" placeholder="请输入用户名"></el-input>
        </el-form-item>
        <el-form-item label="密码" prop="password">
          <el-input type="password" v-model="loginForm.password" placeholder="请输入密码" show-password></el-input>
        </el-form-item>
        <el-form-item>
          <el-button type="primary" @click="handleLogin" :loading="loading" block>登录</el-button>
        </el-form-item>
      </el-form>
    </el-card>
  </div>
</template>

<script setup>
import { ref, reactive } from 'vue'
import { useRouter } from 'vue-router'
import request from '@/utils/request'

const router = useRouter()
const loginFormRef = ref()
const loading = ref(false)

const loginForm = reactive({
  username: '',
  password: ''
})

const rules = {
  username: [
    { required: true, message: '请输入用户名', trigger: 'blur' }
  ],
  password: [
    { required: true, message: '请输入密码', trigger: 'blur' }
  ]
}

const handleLogin = async () => {
  if (!loginFormRef.value) return
  
  try {
    await loginFormRef.value.validate()
    loading.value = true
    
    const res = await request.post('/auth/login', loginForm)
    if (res.token) {
      localStorage.setItem('token', res.token)
      localStorage.setItem('role', res.role)
      
      if (res.role === 'Admin') {
        router.push('/admin/students')
      } else {
        router.push('/student/profile')
      }
    }
  } catch (error) {
    console.error('登录失败:', error)
  } finally {
    loading.value = false
  }
}
</script>
```

[ApiController]
[Route("api/[controller]")]
public class StudentController : ControllerBase
{
    private readonly IStudentService _studentService;
    
    public StudentController(IStudentService studentService)
    {
        _studentService = studentService;
    }
    
    // GET: api/students
    [HttpGet]
    public async Task<ActionResult<IEnumerable<Student>>> GetStudents()
    {
        var students = await _studentService.GetAllStudents();
        return students;
    }
    
    // GET: api/students/{id}
    [HttpGet("{id}")]
    public async Task<ActionResult<Student>> GetStudent(int id)
    {
        var student = await _studentService.GetStudentById(id);
        if (student == null) {
            return NotFound();
        }
        return student;
    }
    
    // POST: api/students
    [HttpPost]
    public async Task<ActionResult<Student>> PostStudent(Student student)
    {
        var newStudent = await _studentService.AddStudent(student);
        return CreatedAtAction(nameof(GetStudent), new { id = newStudent.Id }, newStudent);
    }
    
    // PUT: api/students/{id}
    [HttpPut("{id}")]
    public async Task<IActionResult> PutStudent(int id, Student student)
    {
        if (id != student.Id) {
            return BadRequest();
        }
        
        await _studentService.UpdateStudent(student);
        return NoContent();
    }
    
    // DELETE: api/students/{id}
    [HttpDelete("{id}")]
    public async Task<IActionResult> DeleteStudent(int id)
    {
        var result = await _studentService.DeleteStudent(id);
        if (!result) {
            return NotFound();
        }
        return NoContent();
    }
}
